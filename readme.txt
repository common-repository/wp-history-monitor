Provides a simple interface to monitor modified or created posts, pages or users.

== Description ==

This WP-History-Monitor, monitors your site history like modified dates of posts, pages published, and user's details like when the user was registered and role of the user etc.,


== Installation ==

1. Extract the downloaded zip file.
2. After extracting upload the folder to /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress. That's it.
